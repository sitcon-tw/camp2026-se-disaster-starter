# Prompt：小隊討論

完成三個 persona 訪談、main agent 彙整和自己的取捨決策後，請回到小隊討論。

這一段不是要把大家的答案合併成同一份，也不是要選出誰最好，而是比較不同人如何和 agent 協作。

## 每位學員先用 1 分鐘分享

請每人回答：

1. 哪個 persona 的回饋最有幫助？為什麼？
2. 哪個 persona 的回饋最像在亂給建議？為什麼？
3. main agent 彙整哪裡有幫助？
4. main agent 哪裡過度簡化或太快下結論？
5. 我最後採用了什麼？
6. 我最後拒絕了什麼？

## 小隊共同整理

請小隊一起整理 3–5 點，放到每個人的 `docs/interview-summary.md` 的「小隊討論後的補充」區塊。

可以整理：

- 我們小隊覺得 sub-agent 最有用的地方。
- 我們小隊覺得 sub-agent 最危險的地方。
- 我們小隊看到 main agent 常見的問題。
- 我們下次訪談會問得更好的問題。
- 我們認為不能讓 AI 自動決定的事情。

## 可以請 Codex 幫忙整理，但不要讓 Codex 取代討論

若要請 Codex 協助，可以貼這段：

```text
請根據我們小隊的討論筆記，整理 3–5 點共同觀察。
請不要替我們做產品決策。
請聚焦在：sub-agent 哪裡有用、哪裡危險、main agent 哪裡過度簡化、下次應該怎麼問更好的問題。
```
