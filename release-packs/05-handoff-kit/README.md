# 05 Handoff Kit

這個 release pack 會在變更事件與穩定化後釋出。

目標：讓學員把自己的專案整理給下一位協作者，並接手同隊另一位學員的 repo，體驗軟體生命週期中的交接與維護。

這一階段不是繼續大量加功能，而是檢查：

- 專案是否能被啟動。
- demo 入口是否清楚。
- 需求取捨、流程圖、變更處理是否能被理解。
- 另一位協作者能不能用 Codex 協助理解陌生 repo。
- 哪些文件不足、哪些 AI 產出可能誤導維護者。

## 釋出內容

```text
docs/releases/05-handoff-kit.md
docs/handoff.md
docs/handoff-review.md
docs/showcase-submission.md
docs/prompts/maintainer-onboarding.md
```

## 學員要完成

- 原作者填寫 `docs/handoff.md`。
- 小隊內交換 repo。
- 接手者使用 `docs/prompts/maintainer-onboarding.md` 請 Codex 協助理解陌生 repo。
- 接手者填寫 `docs/handoff-review.md`。
- 每位學員填寫 `docs/showcase-submission.md`，供小隊 showcase 整理素材。
