## 自然語言流程描述

請先用自然語言寫流程，不要一開始就寫 Mermaid。

範例語氣：

```text
原始資訊進來後，資訊整理者先查看來源與原文。
如果資訊不足，標示為需要人工確認。
如果資訊可能誤導行動者，先不要變成候選結果。
如果資訊足夠形成候選結果，建立候選結果，但仍標示為需要人工確認。
每次人工判斷都要留下紀錄。
```

## Mermaid 流程圖

請讓 Codex 根據上面的自然語言描述產生 Mermaid。

請用 VS Code 預覽，確認流程圖能正常顯示。

```mermaid
flowchart TD
  A[原始資訊進入] --> B{資訊是否足夠判斷？}
  B -->|否| C[標示需要人工確認]
  B -->|是| D{是否可以形成候選結果？}
  D -->|否| E[標示不能直接處理]
  D -->|是| F[建立候選結果]
  C --> G[留下判斷紀錄]
  E --> G
  F --> G
```
