# 03 Build Guide

這個 release pack 會在流程設計後釋出。

目標：讓學員根據 `docs/decisions.md` 和 `docs/flow.md`，用 Codex 迭代出 v1 prototype。

這一階段開始可以修改前端程式，但不能回到「請 Codex 隨便做更多功能」的狀態。Codex 應該根據使用者取捨與流程圖實作。

## 釋出內容

```text
docs/releases/03-build-guide.md
docs/build-goals.md
docs/demo-checklist.md
docs/prompts/build-from-flow.md
docs/prompts/review-against-flow.md
```

## 學員要完成

- 一個可以從首頁操作的 v1 prototype。
- prototype 有對應自己的使用者取捨。
- prototype 有對應 `docs/flow.md` 的主流程。
- 畫面看得出哪些資訊仍不確定或需要人工確認。
- `docs/ai-log.md` 有記錄 Codex 如何協助實作。
- 用 `docs/demo-checklist.md` 檢查成果。
